## 2. Gastbeiträge & Content-Kooperationen

### Hochzeitsblogs
Ziel: 2-3 Gastbeiträge pro Quartal auf relevanten Hochzeitsblogs.

**Themenvorschläge:**
- "10 Tipps für die perfekte freie Trauung" (für Hochzeitsblogs)
- "Die schönsten Rituale für eine freie Trauung" (für Lifestyle-Blogs)
- "Destination Wedding Mallorca: Was ihr wissen müsst" (für Reiseblogs)
- "Warum eine freie Trauung immer mehr Paare begeistert" (für Frauenmagazine)

**Kontakt-Vorlage:**
```
Betreff: Gastbeitrag-Idee: [Thema]

Hallo [Name],

ich bin Stefanie, Traurednerin in München (trauworte.com). Euer Blog gefällt mir
besonders wegen [konkretes Beispiel].

Ich hätte eine Idee für einen Gastbeitrag: "[Titel]". Der Artikel würde [kurze
Beschreibung, 2-3 Sätze] und passt perfekt zu euren Leserinnen.

Darf ich euch einen Entwurf schicken?

Herzliche Grüße,
Stefanie Sick
TrauWorte – Traurednerin München
```

### Hochzeitsfotografen-Kooperationen
- Gegenseitige Verlinkung auf "Empfehlungen"-Seiten
- Gemeinsame Styled Shoots → beide verlinken auf ihren Blogs
- Real-Wedding-Features mit Verlinkung

### Wedding Planner Kooperationen
- Empfehlungslisten auf deren Websites
- Gemeinsame Blogbeiträge zu Hochzeitsthemen

## 7. Technische Backlink-Pflege

### Regelmäßig prüfen
- Google Search Console → Links-Bericht
- Broken Links reparieren (301 Redirects)
- Toxic Links identifizieren und disavowen

### Tools (kostenlos)
- Google Search Console
- Bing Webmaster Tools
- Ubersuggest (begrenzt kostenlos)
