# TrauWorte V3 — Recherche, Analyse & Konzept

## 1. RECHERCHE-ERGEBNISSE

### 1.1 Web-Design Trends 2026 (Hochzeitsbranche)

**Visuell:**
- **Serif-Renaissance**: High-contrast Serifs (Didone-Stil) fuer Headlines — wirkt editorial, luxurioes, handgemacht
- **Warm-Earth-Tone Paletten**: Ivory, Champagne, Terracotta, Dusty Rose, Sage — "fuer sich menschlich, nicht hyper-poliert"
- **Unbleached Neutrals**: Off-White statt reines Weiss — waermer, einladender, weniger steril
- **Pantone 2026**: "Cloud Dancer" — soft, airiges Warm-Weiss
- **Texture-Comeback**: Leinen, Grain, Film-Overlays gegen sterile Digital-Aesthetik

**Layout:**
- **Bento-Grid**: Asymmetrische, modulare Card-Layouts (67% der Top-Websites nutzen es)
- **Organic Flow**: Anti-Grid-Design mit fliessenden Linien, weichen Gradienten
- **Soft Maximalism**: Wenige, aber dramatische Elemente (uebergrosse Typo, starke Akzente)
- **Content-First**: Typografie fuehrt, Design unterstuetzt

**Interaktion:**
- **Scroll-Storytelling**: Scrollen wird zur Erzaehlung — Animationen, Parallax, pinned Sections
- **Funktionale Micro-Interactions**: Hover-Effekte, Reveal-Animationen (0.6-1.0s)
- **Human Touch Design**: Handgezeichnete Elemente, echte (nicht Stock-) Fotos, absichtliche Imperfektionen

### 1.3 Zielgruppe: Frauen 25-40

**Fakten:**
- Durchschnittsalter Braut: 28,6 Jahre
- 80% planen hauptsaechlich auf dem Smartphone
- 78% recherchieren Dienstleister online VOR erstem Kontakt
- 77% nutzen Pinterest als Inspirationsquelle
- 69% erwarten starke Online-Praesenz (Website + Social Media)
- Paare machen 80% ihrer Recherche bevor sie jemals Kontakt aufnehmen

**Was konvertiert bei dieser Zielgruppe:**
1. **Emotion vor Information** — "Wie fuehlt sich das an?" vor "Was kostet das?"
2. **Testimonials neben CTAs** — Social Proof am Entscheidungspunkt (+34% Conversion)
3. **Transparenter Prozess** — Was passiert von Anfrage bis Hochzeitstag
4. **Authentische Fotos** — Editorial-Documentary Hybrid, keine Stockfotos
5. **Action-CTAs** — "Verfuegbarkeit pruefen" statt "Kontakt"
6. **Max. 3-4 Formularfelder** — Name, E-Mail, Hochzeitsdatum, Nachricht
7. **Sticky Mobile CTA** — +18% Formular-Submissions

**Farbpsychologie:**
- Warm-Neutrals (Ivory, Beige, Champagne) = Waerme, Komfort, Qualitaet
- Gold/Bronze-Akzente = Luxus ohne Kälte
- Deep Charcoal statt Schwarz = Warm + lesbar
- Warme CTA-Farben schlagen kalte um 21%

## 3. KONZEPT: TrauWorte V3 — "Silk Editorial"

### Design-Philosophie
> "Clean, flowing sections; layered editorial imagery; warm neutral palette with a touch of gold glam. Delicate typography with generous letter-spacing."

**Stil-Name: "Silk Editorial"** — Die Website fuehlt sich an wie ein hochwertiges Hochzeitsmagazin, nicht wie eine Dienstleister-Seite.

### 3.1 Farbpalette (Evolution)

```
FOUNDATION (bleibt):
  --cream:        #FDF4ED   (Hintergrund, Sections abwechselnd)
  --beige:        #FBE9DA   (Hintergrund, Sections abwechselnd)
  --charcoal:     #1a1a1a   (Text, KEIN reines Schwarz)

NEU - AKZENTE:
  --gold:         #B8956A   (CTAs, Highlights, Trennlinien)
  --gold-light:   #D4AF7A   (Hover-States, Gradient-Ende)
  --dusty-rose:   #C4A4A0   (Subtile Akzente, Testimonial-Bereich)
  --sage:         #A8B5A0   (Frische, Natur-Element, sparrisch eingesetzt)
  --deep-brown:   #5C4A3A   (Tiefe, Footer, Kontrast)

TEXTUR:
  --grain-opacity: 0.03     (Subtiler Film-Grain Overlay)
  --paper-texture: url(...)  (Leinen/Papier-Textur auf Sections)
```

### 3.2 Typography (Evolution)

```
HEADLINES:  Boska 400 (bleibt — einzigartiger Markenvorteil)
            + letter-spacing: 0.02em (NEU — "whispers with intention")
            + Groesser: clamp(3rem, 6vw, 5.5rem) fuer H1

BODY:       Inter 400 (bleibt)
            + 17px statt 16px (NEU — komfortableres Lesen)
            + line-height: 1.8 (NEU — mehr Luft)

AKZENT:     Cormorant Garamond Italic (bleibt aus V2)
            Einsatz: Zitate, emotionale Highlights, "Eure Liebe" etc.

LABELS:     Inter 500, 12px, letter-spacing: 0.15em, uppercase
            Einsatz: Section-Labels, Kategorie-Tags
```

### 3.3 Seitenstruktur — Emotionale Storyline

Die Seite erzaehlt eine Geschichte in 10 Akten. Jede Sektion hat eine emotionale Funktion:

```
SEKTION                     EMOTION           ZIEL
─────────────────────────────────────────────────────────
1. Hero (Fullscreen)        Staunen           Atmosphaere setzen
2. Intro-Statement          Vertrauen         Persoenliche Verbindung
3. Social Proof Bar         Sicherheit        Zahlen + Vertrauen
4. Ueber Stefanie           Naehe             Mensch hinter der Marke
5. Prozess/Timeline         Klarheit          Unsicherheit nehmen
6. Angebote (Bento-Grid)    Neugier           Leistungsueberblick
7. Locations                Fernweh           Moeglichkeiten zeigen
8. Testimonials             Bestaetigung      Kaufentscheidung stuetzen
9. Instagram-Feed           Aktualitaet       Lebendigkeit zeigen
10. CTA + Formular          Handlung          Conversion
```

### 3.5 Neue Interaktionen

| Feature | Beschreibung | Prioritaet |
|---|---|---|
| **Scroll-Storytelling** | Sektionen revealen sich beim Scrollen (bestehend + verfeinert) | HOCH |
| **Parallax** | Hero + Locations-Background mit Tiefeneffekt | HOCH |
| **Bento Hover** | Angebote-Cards mit Scale + Shadow + Overlay-Transition | HOCH |
| **CountUp** | Zahlen in Social Proof Bar zaehlen hoch beim Reveal | MITTEL |
| **Testimonial-Karussell** | Swipe/Auto-Rotate (nur wenn >1 echtes Testimonial) | MITTEL |
| **Film Grain** | Subtiler Noise-Overlay (opacity 0.03) auf Sections | NIEDRIG |
| **Cursor-Trail** | Goldener Cursor-Glow auf Desktop (sehr subtil) | NIEDRIG |

### 3.6 Mobile-First Prioritaeten

1. **Sticky CTA-Bar**: Fixierter "Termin anfragen" Button am unteren Bildschirmrand (erscheint nach Hero)
2. **Thumb-Zone Navigation**: Hamburger-Menu mit grossen Touch-Targets (min 48px)
3. **Swipe-Galerien**: Angebote + Testimonials als horizontale Swipe-Karusselle
4. **Vereinfachte Hero**: Bild statt Video, kleinere Typo, CTAs full-width
5. **Core Web Vitals**: LCP < 2.5s, CLS < 0.1, INP < 200ms

### 3.7 Conversion-Optimierung

**CTA-Sprache (NEU vs. ALT):**

| ALT (V1/V2) | NEU (V3) |
|---|---|
| "Jetzt Kontakt aufnehmen" | "Euren Termin anfragen" |
| "Jetzt unverbindlich beraten lassen" | "Verfuegbarkeit pruefen" |
| "Mehr Infos" | "Mehr erfahren →" |
| "Kontakt" (Nav) | "Termin anfragen" (Nav) |

**Social Proof Platzierung:**
- Testimonial-Snippet IN Sektion 4 (Ueber mich) — neben Text
- ★ 5.0 IN Sektion 3 (Social Proof Bar) — oberhalb der Fold-Linie
- Testimonial-Zitat IN Sektion 10 (CTA) — direkt neben dem Formular
- NICHT NUR in einer eigenen Sektion versteckt

## 5. ZUSAMMENFASSUNG

### Warum V3 anders ist als V1/V2:

| Aspekt | V1 (Original) | V2 (Premium) | V3 (Silk Editorial) |
|---|---|---|---|
| Philosophie | 1:1 Nachbau | Premium-Features | Emotionale Storyline |
| Hero | Bild + Text | Cinematic Parallax | Video/Editorial + Emotion |
| Layout | Symmetrisch | Asymmetrisch | Bento + Organic |
| Social Proof | 1 Testimonial | 1 Testimonial | Ueberall verteilt |
| CTAs | Generisch | Generisch | Action-orientiert |
| Prozess | Keiner | Keiner | 4-Schritt Timeline |
| Angebote | 3x2 Grid | 3x2 Grid | Bento Grid |
| Formular | Keines (nur Link) | Keines | Inline (4 Felder) |
| Textur | Keine | Keine | Grain + Paper |
| Instagram | Keines | Keines | Feed-Integration |
| Zielgruppe | Allgemein | Allgemein | Frauen 25-40 |

### Die 3 groessten Differenzierungen:
1. **Emotionale Storyline** statt Feature-Liste — die Seite erzaehlt eine Geschichte
2. **Social Proof ueberall** statt in einer Sektion — Vertrauen am Entscheidungspunkt
3. **"Verfuegbarkeit pruefen"** statt "Kontakt" — action-orientierter Conversion-Flow
